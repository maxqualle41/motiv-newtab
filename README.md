# MotivNewTab

Clean New Tab: To‑Do + Spruch, minimal & stylish

Installation (Entwicklung)

1. Clone oder lade dieses Repo herunter.
2. Öffne Chrome oder Opera und gehe zu chrome://extensions bzw. opera://extensions
3. Aktiviere Entwicklermodus (Developer mode).
4. Klicke auf "Entpackte Erweiterung laden" (Load unpacked) und wähle den Ordner dieses Projekts aus.

Hinweis
- Opera kann das automatische Überschreiben der New‑Tab Seite (chrome_url_overrides) in einigen Profilen blockieren. In diesem Fall funktioniert das Laden in Chrome; alternativ kannst du das Toolbar‑Icon klicken, um die Seite zu öffnen.
